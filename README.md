# Symbi Creature References

Public mirror of canonical creature designs for use as image references in Gamelab Studio (sprite animation pipeline).

Source-of-truth originals live in a private Google Drive. This repo exists purely so external services can fetch reference PNGs via stable public URLs.

Naming: `{biome}_{vitality}_{stage}_{species}.png` for evolved forms,
`{biome}_base_{stage}_{species}.png` for shared eggs/hatchlings.
